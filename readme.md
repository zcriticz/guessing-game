  <h1>Guessing Game</h1>
  <p>In this game, the machine will choose a random number between 0 and 100, and you will have 5 attempts to guess which number it chose. Of course, there will be     some hints: the machine will tell you if the number you entered is greater or less than the number it chose.</p>
